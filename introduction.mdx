## What is ccusage?

ccusage is a CLI tool that analyzes Claude Code token usage and costs from local JSONL files. Track your daily, monthly, and session-based usage with beautiful tables and comprehensive reporting.

## Key features

  <Card title="Daily and monthly reports" icon="calendar">

  <Card title="Session tracking" icon="messages">

## Why ccusage?

- **Ultra-small bundle size** - Incredibly lightweight even without minification, perfect for instant execution with `npx`
- **Offline mode** - Use pre-cached pricing data without network connectivity (Claude models only)
- **Smart tables** - Automatic compact mode for narrow terminals with essential columns
- **Timezone support** - Configure timezone for date grouping with `--timezone` option
- **Locale support** - Customize date/time formatting with `--locale` option (e.g., en-US, ja-JP, de-DE)
- **Configuration files** - Set defaults with JSON configuration files, complete with IDE autocomplete and validation

- **[ccusage](https://www.npmjs.com/package/ccusage)** - Claude Code usage analyzer
- **[@ccusage/codex](https://www.npmjs.com/package/@ccusage/codex)** - OpenAI Codex usage analyzer
- **[@ccusage/opencode](https://www.npmjs.com/package/@ccusage/opencode)** - OpenCode usage analyzer
- **[@ccusage/pi](https://www.npmjs.com/package/@ccusage/pi)** - Pi-agent usage analyzer
- **[@ccusage/amp](https://www.npmjs.com/package/@ccusage/amp)** - Amp usage analyzer
- **[@ccusage/mcp](https://www.npmjs.com/package/@ccusage/mcp)** - MCP server integration
